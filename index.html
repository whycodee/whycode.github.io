Sebelumnya, perkenalkan nama saya Wahyu Saputra. Saya adalah seorang pelajar dan juga penulis konten di YannCode.com. Saya senang mempelajari hal-hal baru dan membagikannya dalam bentuk tulisan. Tujuan dibuatnya blog ini agar ilmu yang saya dapat bisa saya simpan kedalam blog ini dan dapat dibagikan dalam bentuk artikel.

YannCode adalah platform situs pribadi yang dibuat sebagai sarana untuk mengakses informasi dan berbagi ilmu seputar dunia teknologi, programming dan hal-hal bermanfaat lainnya. Platform ini juga sebagai tempat belajar untuk saya pribadi.

YannCode dibuat pada tahun 2021, yaitu tepat saat admin mulai belajar tentang bahasa pemrograman, web dan sebagainya. Hingga saat halaman ini dibuat, admin pun masih dalam tahap belajar untuk menyempurnakan blog ini. YannCode memang merupakan situs pribadi, namun diharapkan kedepannya dapat membuat semua warga internet merasa terbantu dengan adanya situs ini.
